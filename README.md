## Nest-POC 

## Description

- This is a simple rest API with NestJS along side with MongoDB and mongoose

## Installation

- To install NestJS

```bash
$ npm i -g @nestjs/cli
```

- To install mongoose

```bash
$ npm install --save @nestjs/mongoose mongoose
```
- To install MongoDB

  - Follow this link: https://docs.mongodb.com/manual/installation/

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
